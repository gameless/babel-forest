# Babel Forest [![Build Status][status image]][travis]

Not to be confused with [this 2010 movie][forest of babel]. Install [Git] and
[Yarn], then:
```sh
$ git clone https://github.com/samestep/babel-forest.git
$ cd babel-forest
$ yarn
$ yarn start
```
